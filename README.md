# 15 Puzzle Game  
A modern, visually appealing implementation of the classic 15 Puzzle game built with React. Slide tiles in a 4Ã—4 grid to arrange them in numerical order. Solve the puzzle in the fewest moves and time possible.

## Features
- Interactive sliding tile puzzle
- Dynamic win detection with overlay feedback
- Move counter and timer system
- Smooth tile transitions and hover effects

## Installation & Setup

### Clone the Repository
```sh
git clone https://github.com/Vedant-0102/15Puzzle.git
```

Install Dependencies
```sh
npm install
```

Run the Development Server
```sh
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.


## Technologies Used
- **React** (Component-based UI)
- **Vite** (Blazing Fast Build Tool)
- **CSS Modules** (Styling)
